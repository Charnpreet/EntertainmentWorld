## How to build
    1.  Clone the repository
    2.  Install pods
    3.  Open the workspace in Xcode
    4.  Sign up on https://www.themoviedb.org to get Your Key
    5. Go to Utiltiy/Connection and add Your API Key

## Requirements
    Xcode

    UIKit
